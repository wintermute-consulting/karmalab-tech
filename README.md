# karmalab-tech

Static website for [karmalab.tech](https://karmalab.tech), hosted via GitHub Pages.

## GitHub Pages setup

1. Go to **Settings → Pages** in this repository.
2. Under **Source**, select the `main` branch and `/ (root)` folder.
3. Click **Save**. Your site will be live at `https://<org>.github.io/karmalab-tech/` (or your custom domain if configured).
